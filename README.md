# CSS Projects

This workspace contains several small web development exercises focused on HTML and CSS.

## Included Projects

- `bento-box-portfolio` — a portfolio-style page with responsive layout and visual styling.
- `business-card` — a digital business card layout using HTML and CSS.
- `event-player` — a simple event player interface demonstration.
- `greeting-card` — a greeting card design built with CSS.
- `testing` — a sandbox for testing CSS styles and layouts.

## Notes

Each project is self-contained and can be opened in a web browser from its folder.

For best results, use a modern browser and review the related CSS files for styling details.
